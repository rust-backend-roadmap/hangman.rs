# hangman.rs

[Проект виселица](https://zhukovsd.github.io/java-backend-learning-course/Projects/Hangman/).

# Сборка и запуск

```shell
# clone repository
git clone https://github.com/rust-backend-roadmap/hangman.rs

# build executable
cd ./hangman.rs
cargo build --release

# run executable
./target/release/hangmanc
```
